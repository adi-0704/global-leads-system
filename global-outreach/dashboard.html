GitHub Actions Integration

Automate your outreach queue 24/7 in the cloud

To run this system automatically without keeping your computer on, set up these repository secrets on GitHub:

  • SMTP_USER / SMTP_PASSWORD: Sender account (Gmail App Password recommended)
  • IMAP_USER / IMAP_PASSWORD: Email inbox polling credentials to scan replies
  • SMTP_HOST / IMAP_HOST: Server endpoints (default is Gmail)
Scheduled Run: Daily at 09:00 UTC
